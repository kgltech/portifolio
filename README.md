# My Teaching Services — Website

Static 4-page website (Home, About, Services, Contact) for a teacher offering services.

How to use

- Open `index.html` in a browser to view the site.
- The contact form opens your default mail client using `mailto:` (no server required).

Files

- index.html — Home
- about.html — About
- services.html — Services offered
- contact.html — Contact form
- css/style.css — styles
- js/main.js — contact form behaviour
